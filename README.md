## Features

- ⚡️ [Vue 3](https://github.com/vuejs/core) and [Vite](https://github.com/vitejs/vite) for fast development
- 🗂 File-based routing
- 📦 Components auto-importing
- 📑 Layout system
- 🗒 Markdown support
- 🔥 `<script setup>` syntax
- 🖨 Static-site generation (SSG) via [vite-ssg](https://github.com/antfu/vite-ssg)
- 🦾 TypeScript support

<br>
